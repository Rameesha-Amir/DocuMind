# 🧠 DocuMind — AI PDF Chatbot

DocuMind lets you upload any PDF and chat with it using AI. Ask questions, get summaries, and extract key insights instantly.

## Features
- Upload any PDF
- Ask questions in natural language
- Powered by Google Gemini AI
- Quick buttons: Summary, Key Points, Overview, Topics
- Clean dark UI — no installation needed

## Tech Stack
- HTML, CSS, JavaScript (single file)
- PDF.js • Mammoth.js
- Google Gemini API — AI responses

## API Used
Google Gemini API — gemini-2.5-flash

## How to Run
1. Download the repository
2. Open index.html in browser
3. Upload a PDF and start chatting!
